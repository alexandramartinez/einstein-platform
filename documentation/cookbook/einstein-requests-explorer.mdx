# Einstein Requests Explorer

Instroducing the Einstein Requests Explorer, the most accurate way to estimate AI usage in your Salesforce org. Whether you're a developer fine-tuning your AI integrations or an admin keeping an eye on usage limits, we've got you covered.

The tool offers several helpful features:

- An interactive calculator where you input a token count and see the exact number of Einstein Requests consumed.

- A token counter that lets you see exactly how many tokens are consumed by a prompt for a given model. A word count is also provided to help you understand the size of your prompt.

- Ready-to-use formulas for Excel or Google Sheets.
